## 🚀 Deploy to GitHub Pages (5 minutes)

### Step 1: Create GitHub Account
1. Go to https://github.com
2. Sign up for a free account

### Step 2: Create Repository
1. Click the "+" icon (top right) → "New repository"
2. Name it: `elitez-catalogue`
3. Make it **Public**
4. Click "Create repository"

### Step 3: Upload Files
1. On your repository page, click "uploading an existing file"
2. Drag and drop the ENTIRE `elitez-catalogue` folder contents:
   - All .html files
   - The `images` folder
   - The logo.png file
3. Scroll down, add commit message: "Initial catalogue upload"
4. Click "Commit changes"

### Step 4: Enable GitHub Pages
1. Go to repository Settings (top menu)
2. Click "Pages" in the left sidebar
3. Under "Source", select "main" branch
4. Click "Save"
5. Wait 2-3 minutes for deployment

### Step 5: Get Your URL
Your catalogue will be live at:
```
https://YOUR-USERNAME.github.io/elitez-catalogue/
```

## 📝 How to Edit Later

### Add New Products:
1. Go to your repository on GitHub
2. Navigate to the sport page file (e.g., `rugby-league.html`)
3. Click the pencil icon (Edit)
4. Copy/paste an existing product card HTML block
5. Update the image filename and product name
6. Commit changes

### Add New Images:
1. Click into the `images` folder
2. Click "Add file" → "Upload files"
3. Drag your PNG images
4. Commit changes

### Change Text/Prices:
1. Click any .html file
2. Click pencil icon to edit
3. Make your changes
4. Commit changes
